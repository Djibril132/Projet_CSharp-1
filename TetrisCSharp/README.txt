--- Requis
    -- DotNet


--- How to start
    -- Il suffit d'ouvrir l'invite de commande dans le dossier TetrisCSharp et d'écrire "dotnet run"


--- Description des fichies

    -- Calculate.cs
        - Utilisé pour les méthodes de calcul comme les collisions et les rotations

    -- Draw.cs
        - Utilisé pour dessiner l'écran de jeu

    -- Program.cs
        - Contient les variables et la loop entière du jeu


--- Les touches

    -- Espace
        - Lancer le jeu dans le menu général
        - Relancer une partie dans le menu game over
        - Placer la piece directement dans le jeu

    -- S
        - Tourner la piece vers la gauche

    -- D et flèche du haut
        - Tourner la piece vers la droite

    -- flèche de gauche
        - Déplacer la piece vers la gauche
    
    -- flèche de droite
        - Déplacer la piece vers la droite

    -- flèche du bas
        - Déplacer la piece vers le bas

    -- C
        - Mettre la piece de côté

    -- Echap
        - Fermer le jeu
